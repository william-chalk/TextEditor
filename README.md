# TextEditor

# User Story
```
AS A developer
I WANT to create notes or code snippets with or without an internet connection
SO THAT I can reliably retrieve them for later use
```

# Installation
```
npm i
npm run build
```

# Usage

A simple text editor that a user can use to write notes or save a text snippet
